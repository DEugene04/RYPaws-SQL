# Project Title
RYPaws SQL Analysis

# Problem
Currently, RYPaws Pet Shop is still using manual management system to maintain the sale and purchase. You as a professional database administrator and a good friend want to help RYPaws Pet Shop to create a database system so that Andra and his brother don’t have to use the old manual management system anymore.

# Tools
- SQL
- SSMS
